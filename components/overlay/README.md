# Overlay Yanapay

Agente flotante de Electron. Muestra y pronuncia consejos recibidos por SSE desde
el servidor local.

- `Ctrl+Shift+Espacio`: escuchar una pregunta.
- Palabra de activacion `Yanapay`: escucha automatica sin botones.
- La voz se reconoce localmente mediante las capacidades de voz instaladas en Windows.

Para español, instala un paquete de voz/reconocimiento desde Configuracion de
Windows > Hora e idioma > Voz.
