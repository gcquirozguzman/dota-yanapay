# Voz local

Listener continuo basado en Faster Whisper. Detecta automáticamente actividad de
voz, transcribe en español y emite preguntas que contienen Yanapay o variantes
fonéticas como Janapai. El audio no sale del equipo.

Prueba con una grabación:

```powershell
npm run voice:test-audio -- "audio/20260712/Grabación.m4a"
```
